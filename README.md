# Check Image Text App

画像に含まれるキーワードにマッチするかチェック

画像ファイルは「./public/checkImage」へ格納

## npm
モジュールをインストール
```shell
npm install
```
アプリケーション実行
```shell
gulp
```

キーワードを入力して「Check Text」
